# Foton OS v2 — Visual Identity + New Modules Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate Foton OS from dark-navy to white/Foton-blue visual identity and integrate BP Tracker, Field App, and Dirección Panel as connected Supabase-backed routes.

**Architecture:** Static HTML files in `public/` served from Netlify. No build tools. Each page is self-contained HTML with inline CSS/JS and a shared `_config.js` for Supabase credentials. Supabase JS and Chart.js loaded via CDN.

**Tech Stack:** Vanilla HTML/CSS/JS, Supabase JS v2 (CDN), Chart.js v4 (CDN), jsPDF v2.5 (CDN), Netlify static hosting.

// ── Load BP + facturado from Supabase ────────────────────
async function loadFromSupabase() {
  if (!db) return;
  try {
    const { data: bpRows } = await db
      .from('bp_anual')
      .select('distribuidor_id,modelo,unidades_bp')
      .eq('anio', 2026);

    const { data: facRows } = await db
      .from('facturado_ytd')
      .select('distribuidor_id,mes,modelo,unidades_facturadas')
      .eq('anio', 2026);

    if (!bpRows || !facRows) return;

    // Update BP targets in the existing data structure
    bpRows.forEach(row => {
      const d = DEALERS[row.distribuidor_id];
      if (d && d.bp2026 && d.bp2026[row.modelo] !== undefined) {
        d.bp2026[row.modelo] = row.unidades_bp;
      }
    });

    // Build facturado totals per dealer (sum all months all models)
    facRows.forEach(row => {
      const d = DEALERS[row.distribuidor_id];
      if (d) {
        if (!d._facturadoDB) d._facturadoDB = {};
        if (!d._facturadoDB[row.modelo]) d._facturadoDB[row.modelo] = 0;
        d._facturadoDB[row.modelo] += row.unidades_facturadas;
      }
    });

    console.log('[Foton OS] BP data loaded from Supabase');
    renderCurrentDealer();
  } catch (e) {
    console.warn('[Foton OS] Supabase load failed, using local data:', e);
  }
}

// ── Save facturado edit to Supabase ─────────────────────
async function saveFacturadoToSupabase(dealerId, modelo, unidades) {
  if (!db) return;
  const mesActual = new Date().getMonth() + 1;
  await db.from('facturado_ytd').upsert({
    distribuidor_id: dealerId,
    anio: 2026,
    mes: mesActual,
    modelo: modelo,
    unidades_facturadas: unidades,
    updated_at: new Date().toISOString()
  }, { onConflict: 'distribuidor_id,anio,mes,modelo' });
}

    if (semClass === 'sem-red') fillColor = '#C0392B';

  });
